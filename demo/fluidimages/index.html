图片自适应高宽方案

方案 1

高宽设置为100%，图片将按父级元素长宽拉伸或缩小，图片失真

li {
    width: 250px;
    height: 250px;
}
img {
    height: 100%;
    width: 100%;
}

方案 2

高宽设置为指定max值，图片将按原始等比缩小，图片不失真

li {
    width: 250px;
    height: 250px;
}
img {
    max-width: 250px;
    max-height: 250px;
}

IE7+ 支持 max-width/max-height, IE6- 不支持

IE6 下可以使用 css expression 或 js 方式, 避免复杂化，推荐css expression，可忽略性能问题

  <!--[if lte IE 6]><style>
    li img{
        width: expression(this.width > 250 ? 250+"px" : "auto" );
    }
</style><![endif]-->

方案 3

父级设置overflow：hidden，img高宽设置为auto，图片按源像素大小显示，图片不失真，超出指定区域则不显示

li {
    width: 250px;
    height: 250px;
    overflow: hidden;
}
img {
    height: auto;
    width: auto;
}
